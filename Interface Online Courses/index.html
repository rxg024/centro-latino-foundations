About Interface Web School

Our story

Interface Web School is located in Omaha, Nebraska and has been teaching web technology courses live and in-person since 2014. As a startup, we were free to develop our personal style of instruction that emphasizes the practical topics and experiences to help foster a deeper understaing of tech that goes beyond just the syntax.

In 2017 Interface was acquired by the AIM Institute. AIM's mission is to grow, connect and inspire the tech talent community through educational and career development initiatives. Our focus is to get people excited about technology, and inspire them to purse a tech career - which leads to a better work force and ultimately allos communities to thrive.

FAQ

Courses

Do I need experience?
For the Basics of Web Development course, no - this course will teach you everything you need to know. For the Mobile Wed Development Course, yes - you should have a solid understanding of both HTML & CSS, including using advanced selectors like nth-of-type and :target.
Do I need to install any software?
You only need Chrome - everything will be presented online, and you will use our online development plattform for all coding work.
Is this course hard?
Maybe - kinda depends on you! The material is presented in a very step-by-step manner as it guides you through new topics while you are developing a real website. Remmember that these are also self-paced courses, so if you need to review a section multiple times to really "get-it", you can.
How long is this course?
Courses are open from the first of the month for 31 days, and each course is divided into about 8-10 sessions, so you typically have about 3 days on average to complete each section. Each section can take anywhere from 1-4 hours to complete.
